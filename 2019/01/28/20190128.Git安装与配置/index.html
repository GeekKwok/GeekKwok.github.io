六、本地项目初始化及提交Git仓库

  1. 打开 IDEA，配置好 JDK、Maven、KeyMap 等，然后通过 maven archetype webapp 新建空项目

  2. 等 Maven 工程初始化完成后，在 IDEA 内打开 Terminal 初始化 README.md 和 .gitignore 文件

  3. 编辑 .gitignore 文件

  4. 新建 java、test 等目录并做标注

  5. 在项目对应目录下初始化 git 仓库

  6. 添加项目内文件到本地仓库

  7. 在 gitee.com 上新建一个 mmall 仓库并与本地仓库关联

  8. 创建 v1.0 分支并推送到远程，v1.0 分支开发，master 主分支发布
